
- 一个原生左右切换式轮播图

[效果展示](https://hjy0810.github.io/demo/swiper/)

[查看代码](https://github.com/hjy0810/demo/tree/master/swiper)

#### 功能
```
* 自动轮播，焦点跟随
* 点击左右箭头实现左右切换，连续点击时使用flag标识避免显示混乱，当前图片切换到下一张之前点击无效
* 点击底部焦点可以跳转到对应的图片
* 鼠标移入，停止播放；鼠标移出，继续播放
```

#### 实现思路
* 为了实现无缝切换，将图片的第一张复制一份到图片列表的最后
* 轮播图盒子的宽度相对自身定位，通过控制left属性进行左右轮播
* 通过全局变量index控制轮播图播放，并进行左右边界判断，当轮播到位于最后位置处的伪第一张图片，将要进行下一次轮播之前，通过设置轮播图盒子left属性让其立即回到真实的第一张，然后开始向右运动；当位于第一张图片，点击向左运动的箭头，通过设置轮播图盒子left属性立即让其回到最后位置处的伪第一张图片，然后开始向左运动。
* 快速点击左右箭头，为了避秒轮播图显示混乱，通过设置flag，保证在当前图片切换正确切换到下一张之间的点击无效。
* 鼠标移入，清除自动轮播定时器，鼠标移出，autoPlay方法执行即可
* 为了保证第一个焦点在第一次进入页面时能正确显示样式，默认第一个焦点有current样式类，之后无论自动轮播、点击焦点、点击左右箭头，都通过全局变量index控制焦点的样式。只要图片切换，首先清空左右焦点的current样式类，然后根据index给当前图片对应的焦点添加current样式类名
* 进入play函数，index首先自动加一，所以当点击左箭头，index-=2，然后让play执行；当点击右箭头，直接play执行即可
